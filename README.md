# 💈 Barber Style

Site moderno de barbearia desenvolvido com HTML e CSS, com visual premium neon dourado, animações suaves e design responsivo.

---

# ✨ Preview

O projeto possui:

- Header fixo animado
- Efeito neon dourado
- Fundo animado
- Cards interativos
- Página de cortes
- Página sobre
- Página equipe
- Animações modernas
- Responsividade
- Efeitos visuais premium

---

# 📸 Páginas

## 🏠 Home
Página inicial da barbearia.

## ✂️ Cortes
Galeria de cortes modernos com cards animados.

## 👨‍🔧 Equipe
